git status - check latest
git add . - add all in the red
git rm --cached main.js - remove main js
git add *.html - get all .html files
git commit > press I for insert mode > press escape to exit > then press wq

open gitignore and type the exact name file of the document that you don't want to include

git branch login - create branch and 'login' is the variable name

git checkout branch - to switch branch
git checkout master - to switch back , you will lose all your created files once you switch back
git merge login - to merge 2 master and sub branches login is the variable name
